# World Map Web Application

This is a web application that allows users to search for weather information and popular places around the world. The application utilizes Leaflet.js, an open-source JavaScript library for interactive maps, and various APIs to fetch weather and location data.

## Features

- Search for weather information of a specific city
- Search for popular places in a city
- Search for country details, including capital, population, area, currency, and languages

## Getting Started

To run the application locally, follow these steps:

1. Clone the repository or download the HTML file.
2. Open the HTML file (`index.html`) in a web browser.
3. Use the search inputs and buttons to interact with the application.

## APIs Used

- Weather API: [Weatherbit](https://www.weatherbit.io/)
- Geocoding API: [OpenCageData](https://opencagedata.com/)
- Country Information API: [REST Countries](https://restcountries.com/)
